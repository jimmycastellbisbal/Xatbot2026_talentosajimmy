# Biblioteca Eco-Digital - Assistent Virtual Intel·ligent

Benvingut al repositori de l'Assistent Virtual Oficial per a l'entorn sostenible de Can Font. Aquest és un projecte col·laboratiu de grup desenvolupat per al mòdul de SMX.

L'objectiu principal és oferir un xat interactiu que resolgui dubtes sobre la instal·lació d'una manera 100% digital, seguint la política ambiental de **Residu Zero** (prohibició d'imprimir en paper).

## 🚀 Característiques Tècniques
- **Model d'IA:** Gemini 2.5 Flash (Google GenAI SDK Nativa)
- **Entorn d'execució:** Google Colab / Python
- **Seguretat:** Gestió de claus privades mitjançant Colab Secrets (UserData)
- **Persistència:** Context persistent de diàleg amb `client.chats.create()`

## 🛠️ Instruccions d'Instal·lació i Ús
1. Accedeix al fitxer `Chatbot_eco_digital.py` des de Google Colab.
2. Activa la icona de la clau (**Secrets**) al menú lateral de Colab.
3. Afegeix una nova variable anomenada `GOOGLE_API_KEY` i introdueix-hi la teva clau privada d'AI Studio.
4. Executa totes les cel·les de codi.
5. Escriu el teu nom quan el terminal ho sol·liciti i interactua amb l'assistent. Escriu `Sortir` per finalitzar.
